# Sitio Web Estático – Landing de Turismo (Proyecto Académico)

Proyecto académico realizado como práctica inicial de desarrollo web.

Este repositorio se publica como parte del recorrido formativo y representa una aplicación web básica desarrollada en los comienzos del aprendizaje.

---

## Descripción

El proyecto consiste en un sitio web estático de tipo landing page para una empresa ficticia de turismo.

Incluye:
- múltiples páginas HTML,
- navegación entre secciones,
- estilos aplicados con CSS,
- estructura básica de un sitio informativo.

El objetivo principal fue practicar maquetación, estructura HTML y estilos CSS.

---

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript (básico)
